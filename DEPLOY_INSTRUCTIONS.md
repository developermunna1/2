# Render এ প্রজেক্ট হোস্ট করার নিয়ম (Headless Bot)

আপনার প্রজেক্টটি Render এ সফলভাবে রান করার জন্য আমরা `Docker` ব্যবহার করব, কারণ Chrome ব্রাউজার ইন্সটল করার জন্য এটি সবচেয়ে সহজ উপায়।

নিচের ধাপগুলো অনুসরণ করুন:

১. **GitHub এ কোড আপডেট:**
   - আমি প্রয়োজনীয় ফাইল (`Dockerfile`, `headless_run.py`, `requirements.txt`) তৈরি করে দিয়েছি।
   - GitHub এ কোডগুলো অটোমেটিকভাবে আপলোড হয়ে যাবে।

২. **Render এ নতুন সার্ভিস তৈরি:**
   - [render.com](https://render.com) এ যান।
   - "New +" -> **"Web Service"** সিলেক্ট করুন। (Note: Bot টি যদি ব্যাকগ্রাউন্ডে সবসময় চালাতে চান তবে "Background Worker" ভালো, কিন্তু ফ্রি টায়ারে "Web Service" ও ব্যবহার করা যায়)।
   - আপনার GitHub রিপোজিটরি (`developermunna1/2`) কানেক্ট করুন।

৩. **কনফিগারেশন (Configuration):**
   - **Name:** সার্ভিসের নাম দিন।
   - **Region:** `Singapore` বা সুবিধামতো দিন।
   - **Branch:** `master`।
   - **Runtime:** **Docker** সিলেক্ট করুন (খুবই গুরুত্বপূর্ণ!)।
     - (আগে Python ছিল, এখন Docker সিলেক্ট করতে হবে কারণ আমরা Dockerfile ব্যবহার করছি)।
   - **Instance Type:** Free (বা আপনার প্রয়োজনমতো)।

৪. **Advanced (Optional):**
   - যদি `namber.text` ফাইলে অনেক নাম্বার থাকে, তবে নিশ্চিত করুন ফাইলটি ঠিকঠাক আছে।

৫. **Deploy Service:**
   - "Create Web Service" বাটনে ক্লিক করুন।
   - Render এখন Docker Image তৈরি করবে। এটি প্রথমবার ৫-১০ মিনিট সময় নিতে পারে।

৬. **লগ চেক করা:**
   - ডিপ্লয় হয়ে গেলে "Logs" ট্যাবে দেখবেন:
     - `Starting Headless Bot...`
     - `Loaded X numbers.`
     - `[BOT LOG] ...`

**দ্রষ্টব্য:**
- যেহেতু এটি একটি Bot এবং কোনো ওয়েবসাইট দেখাচ্ছে না, তাই Render এর লিংকে ক্লিক করলে কিছু দেখাবে না বা Error দিতে পারে। কিন্তু "Logs" এ চেক করলে দেখবেন কাজ হচ্ছে।
- ভুলবশত `. (dot)` কমান্ড দিবেন না। Docker সিলেক্ট করলে কোনো কমান্ড লিখতে হবে না, সব অটোমেটিক হবে।

কোনো সমস্যা হলে জানাবেন!
